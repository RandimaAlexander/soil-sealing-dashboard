# 🌍 Global Soil Sealing Dashboard

An interactive Streamlit dashboard developed to analyse soil sealing trends across countries and regions. The dashboard provides insights into environmental impact, urbanisation patterns, and sustainability risks using data visualisation techniques.

## 🎯 Project Overview

This project focuses on analysing soil sealing data obtained from the European Environment Agency. Soil sealing is a key indicator of environmental change, representing land covered by impermeable surfaces such as concrete and asphalt.

The dashboard enables users to:
- Explore trends over time  
- Compare regions  
- Identify high-growth countries  
- Analyse environmental impact
- 
## Features

-  Interactive filters (Country, Region, Year range)  
-  Trend analysis over time  
-  Regional comparison  
-  Growth ranking of countries  
-  Scatter plot for relationship analysis  
-  Distribution analysis  
-  Dynamic insight generation
